Version 1.0.beta

The EA:Ragnarök Schwarmflotte Dagon Tyranid list is heavily influenced by the F-ERC Tyranids, in particular the army structure, but with plenty of inspiration drawn from the NetEA und E-UK versions as well. Some key aspects are the absence of a Swarm rule (though this could appear in other Tyranid lists), und an adapted version of the Super-Entbehrlich rule from E-UK.

Ein of the main features of the list is that Synapse creatures are usually possible to target separately from the surrounding formation, making it possible to decapitate these Verbände, giving them a number of penalties to activation etc. Such 'headless' swarms can be absorbed into other swarms with remaining synapse creatures.

The structure of the Dagon Tyranid list is eine of its main features, making it possible to be very creative with formation composition. It has been a challenge to make it attractive to not only take basic Assault Swarms, though that is certainly a viable army to take.
